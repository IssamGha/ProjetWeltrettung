#  Weltrettung – MVC Webanwendung

##  Projektbeschreibung
Die Anwendung „Weltrettung“ ist eine ASP.NET Core MVC Web-App zur Verwaltung von Weltrettern, Bedrohungen und Aggressoren.

Ziel ist die Umsetzung einer vollständigen CRUD-Webanwendung mit Formularen, Datenverwaltung und Entitätsmodellierung.

---

##  Funktionen

###  Weltretter-System
- Registrierung von Weltrettern
- Vorname, Nachname, E-Mail
- Optional: Fähigkeit
- Altersprüfung (18+ Checkbox)
- Liste aller Weltretter

###  Bedrohungs-System
- Verwaltung von Aggressoren
- Erstellung von Bedrohungen
- Verknüpfung zwischen Aggressor und Bedrohung
- Optional: Held-Zuordnung

###  Allgemein
- Startseite mit Weltkarte
- Navigation mit Rücksprung zur Startseite
- CRUD-Funktionalität
- Repository (In-Memory Speicherung)

---

##  Technologien
- ASP.NET Core MVC
- C#
- HTML / CSS
- Bootstrap
- Repository Pattern

---

2. In Visual Studio öffnen
3. Abhängigkeiten installieren
