#  Hochwärtsbibliothek (ASP.NET Core MVC)

##  Beschreibung
Webanwendung zur digitalen Verwaltung einer Bibliothek.  
Bücher können erstellt, verliehen, zurückgegeben und überprüft werden.

##  Funktionen
- Bücher verwalten (CRUD)
- Verleih- und Rückgabesystem
- Statusverwaltung (verfügbar, verliehen, verschollen)
- Beschädigungsprüfung
- Datenbankanbindung (Entity Framework Code-First)

##  Technologien
- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- HTML / Bootstrap

##  Start
1. Repository klonen  
2. Datenbank konfigurieren  
3. Migration ausführen:
   ```
   Update-Database
   ```
4. Anwendung starten
